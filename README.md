# SafeCard 防災避難聯絡卡產生器

台灣地震頻繁，SafeCard 讓你在瀏覽器填好資料後，直接列印成可放進避難包的小卡。

**[立即使用 →](https://legendream.github.io/safecard/)**

---

## 功能

### 緊急聯絡小卡（名片尺寸 90 × 54 mm）
- **正面**：姓名、血型、出生年月日、身分證字號、過敏史、病史
- **背面**：緊急聯絡人（含關係）× 2、主要與次要避難地點

### 寵物個資與健康資料卡（A7 尺寸 74 × 105 mm）
- **正面**：名字、物種、性別、品種、生日、外觀描述、照片區、身份辨識（晶片 / 名牌）、用藥（慢性病 × 2 各自對應用藥）
- **背面**：照顧者聯絡資訊、健康管理（絕育、疫苗、藥物過敏）、飲食習慣（各餐次食物與份量、注意事項）、就診診所

---

## 隱私設計

- **零後端**：純靜態 HTML 單一檔案，不需伺服器、不需安裝
- **零儲存**：所有資料只存在於瀏覽器記憶體，關閉或重整即消除
- **零外連**：無 CDN、無分析工具、無任何外部請求
- **離線可用**：下載 `index.html` 後可完全離線使用

---

## 使用方式

### 線上版
直接開啟 [https://legendream.github.io/safecard/](https://legendream.github.io/safecard/)

### 離線版
1. 下載 [`index.html`](./index.html)
2. 用瀏覽器開啟（不需伺服器）
3. 填寫資料 → 點擊「列印 / 下載紙本」

### 列印設定
瀏覽器列印對話框請確認：
1. 紙張尺寸設為 **A4**
2. 縮放比例設為 **100%**（勿勾選「符合頁面」）
3. 建議開啟「背景圖形」以保留色塊
