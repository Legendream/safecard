# SafeCard 防災避難聯絡卡產生器

地震、颱風、火災或任何意外都適用。SafeCard 讓你在瀏覽器填好資料後，直接列印成可放進避難包的小卡——當你走散、受傷說不出話，或得緊急託付動物夥伴時，這張隨身小卡會替你回答「你是誰、該聯絡誰、有哪些過敏與病史」。

**[立即使用 →](https://safecard.claire-cheng.com/)**

---

## 功能

提供三種卡片，從封面選擇要製作哪一張，左側即時預覽、右側填寫，可切換正面／背面預覽：

### 01 避難聯絡卡（名片尺寸 90 × 54 mm，核心卡）
- **正面**：大頭照、姓名、出生年月日、身分證字號、過敏史、病史、急救指示
- **背面**：緊急聯絡人（含關係）× 2、主要與次要避難地點

### 02 醫療資訊卡（名片尺寸 90 × 54 mm，補強卡）
- 給有慢性病、過敏或需長期用藥的人
- **欄位**：血型、藥物／食物過敏、慢性病與醫療狀況、目前用藥、急救指示、體內植入物、器官捐贈意願、健保卡號、主治醫師與醫院聯絡

### 03 動物夥伴卡（A7 尺寸 74 × 105 mm，專用卡）
- **正面**：名字、物種、性別、品種、生日、外觀描述、照片區、身分辨識（晶片 / 名牌）、用藥狀況（可動態新增多筆）
- **背面**：照顧者聯絡資訊、健康管理（絕育、疫苗、藥物過敏）、飲食習慣（可動態新增多餐次）、就診診所

### 易用性設計
- **封面引導**：進站先看封面，依「要幫誰準備」推薦適合的卡片
- **字級切換**：小／中／大三段，按鈕本身以大小示意級距
- **實際尺寸預覽**：以信用卡／悠遊卡校正後，可 1:1 預覽實際印刷大小
- **行動版底部抽屜**：手機上以浮動鈕呼叫預覽卡片，並可直接列印

## 隱私設計

- **零後端**：純靜態 HTML 單一檔案，不需伺服器、不需安裝
- **零儲存**：所有資料只存在於瀏覽器記憶體，關閉或重整即消除
- **零外連**：無 CDN、無分析工具、無任何外部請求
- **離線可用**：下載 `index.html` 後可完全離線使用

## 使用方式

### 線上版
直接開啟 [https://legendream.github.io/safecard/](https://legendream.github.io/safecard/)

### 離線版
1. 下載 [`index.html`](./index.html)
2. 用瀏覽器開啟（不需伺服器）
3. 填寫資料 → 點擊「列印 / 下載紙本」

### 列印設定
瀏覽器列印對話框請確認：
1. 紙張尺寸設為 **A4**
2. 縮放比例設為 **100%**（勿勾選「符合頁面」）
3. 建議開啟「背景圖形」以保留色塊
